# MysqlRecovery
Mysql binlog delete 语句恢复，支持超大文件转换。


例如如下语句：

```
### DELETE FROM `sysdb`.`base_log`
### WHERE
###   @1='53a2764b-55b3-4fea-a9a9-8c3656577a09'
###   @2=4.000000000000000000000000000000
###   @3=NULL
###   @4=NULL
###   @5='2020-04-07 23:36:52'
###   @6='System'
###   @7=''
###   @8='9'
###   @9=''
###   @10=NULL
###   @11=NULL
###   @12='10.86.8.236'
###   @13=NULL
###   @14='DESKTOP-BVG70RT'
###   @15='Chrome 70.0'
###   @16=-1.000000000000000000000000000000
###   @17=''
###   @18=NULL
###   @19=0.000000000000000000000000000000
###   @20=1.000000000000000000000000000000
```

将转换为 insert 语句
